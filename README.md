 # Real-Time Boxing Punch Detection using Computer Vision

## Overview
Developed a real-time computer vision system to detect and classify boxing punches (jab, cross, hook, uppercut).

## Tech Stack
- Python
- OpenCV
- CNN / Deep Learning

## Features
- Real-time inference
- Multi-class classification
- Video-based detection pipeline

## Results
- Accuracy: 87%
- Real-time performance: 60 FPS

## Applications
- Sports analytics
- Athlete performance tracking
